# Лабораторні роботи з предмету "Алгоритми та структури даних".

Репозиторій для лабораторних робіт з курсу "Алгоритми та структури даних".
Тут знаходяться всі лабораторні завдання курсу, згруповані по папках за номерами.

## Зміст

| Назва лабораторної роботи | Папка |
|---|---|
| Лабораторна робота №1. Структура даних стек та основні операції на стеках | lr1 |
| Лабораторна робота №2. Структура даних черга, основні операції | lr2 |
| Лабораторна робота №3. Структура даних геш-таблиця, основні операції | lr3 |
| Лабораторна робота №4. Структура даних бінарне дерево, операції на структурі даних | lr4 |
| Лабораторна робота №5. Сортування вставками | lr5 |
| Лабораторна робота №6. Сортування злиттям | lr6 |
| Лабораторна робота №7. Сортування купою | lr7 |
| Лабораторна робота №8. Швидке сортування | lr8 |
| Лабораторна робота №11. Алгоритм Крускала | lr11 |
| Лабораторна робота №12. Алгоритм Прима | lr12 |
| Лабораторна робота №14. Комбінаторні завдання | lr14 |
| Лабораторна робота №15. Лінійні та нелінійні задачі програмування | lr15 |

## Інструкції

1. Кожна лабораторна робота зберігається у відповідній папці (`lr1`, `lr2`, ...).
2. Для перевірки роботи відкрийте відповідну папку та знайдіть файл з рішенням.
